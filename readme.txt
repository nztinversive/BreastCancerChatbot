# Breast Care Navigator Assistant (BCNA)

## Overview
The Breast Care Navigator Assistant (BCNA) is an AI-powered chat interface designed to provide information and guidance about breast cancer care. This project aims to connect patients with the right level of care based on their needs, offering support for questions about diagnosis, treatment options, and the care process.

## Current State
This project is in its initial development stage. It currently includes:
- A frontend chat interface with a modern, purple design
- A basic Flask backend to serve the interface and handle messages
- Placeholder responses (to be replaced with AI-generated content in future iterations)

## Features
- Clean, user-friendly chat interface
- Real-time message exchange
- Responsive design for various screen sizes

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask

## Setup and Installation

### Prerequisites
- Python 3.7+
- pip (Python package manager)

### Steps
1. Clone the repository:
   ```
   git clone https://github.com/your-username/bcna-chat.git
   cd bcna-chat
   ```

2. Create a virtual environment (optional but recommended):
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
   ```

3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

4. Run the Flask application:
   ```
   python app.py
   ```

5. Open a web browser and navigate to `http://localhost:5000` to view the application.
